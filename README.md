## 🚀 Features

✨ **Domain File Scanner**
- Scan bulk domains with HTTP/HTTPS
- Detect status codes, IP, and server headers
- Resume scanning support
- IP blocklist feature

⚡ **Advanced TCP + HTTP Scanner**
- CIDR scanning (e.g. 1.1.1.0/24)
- IP list scanning
- TCP port detection
- HTTP/HTTPS HEAD requests
- Server fingerprinting:
  - CloudFront
  - Google
  - Vercel
  - Caddy

🔍 **Domain Extractor**
- Extract domains by keyword
- Smart filtering (no unnecessary subdomains)

🧹 **Domain Cleaner & Filter**
- Remove duplicates
- Remove invalid domains
- Clean URLs → pure domain format
